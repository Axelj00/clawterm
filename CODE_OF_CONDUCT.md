# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at the link above. In short: be respectful, be constructive, be welcoming.

## Reporting

If you experience or witness unacceptable behavior, please email the maintainer directly. Reports will be handled confidentially.

## Enforcement

Project maintainers who do not follow or enforce the Code of Conduct may face temporary or permanent repercussions as determined by other members of the project's leadership.
